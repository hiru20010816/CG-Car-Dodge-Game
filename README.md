# Car Dodge Game

This is a simple 2D car dodge game developed for the Computer Graphics module.

## Game Concept

The player controls a car on a road and avoids enemy cars coming from the top. The score increases when the player successfully avoids obstacles. The game ends when the player collides with an enemy car.

## Controls

- Left Arrow / A: Move left
- Right Arrow / D: Move right
- R: Restart
- ESC: Exit

## Main Features

- 2D road rendering
- Player car movement
- Enemy car movement
- Collision detection
- Score system
- Game over screen
